Case Study: StarWorm Art

Overview

Problem: Many local artists struggle to showcase their work online without expensive platforms or galleries.
Solution: I designed and developed a responsive portfolio template specifically for independent artists, enabling them to easily share and organize their work online.

Audience

This project is for independent visual artists, illustrators, and designers who want a clean, minimal, and customizable digital presence without relying on third-party platforms like Etsy or Instagram alone.

Process

Sketches & Wireframes

I started by sketching multiple layout options and tested them with peers. From there, I moved to wireframes in Figma to structure the layout hierarchy and user flow.

Mockups

Once structure was approved, I designed high-fidelity mockups using a clean palette of indigo and soft violet with Montserrat + Roboto for a professional, artsy tone.

User Testing

I tested the prototype with 3 local artists. Feedback focused on clearer CTAs and larger gallery thumbnails. I incorporated changes before development.

Tech Stack

  • HTML5 for structure
  • CSS + Bootstrap 5 for responsive layout and styling
  • Figma for design, layout, and prototyping
  • JavaScript (minor) for interactivity

Reflection

This was one of the most rewarding projects I’ve worked on. It balanced both design and development and served a real audience. I learned how to iterate quickly with user feedback, improve mobile responsiveness, and build with reuse in mind using Bootstrap components.

In the future, I’d like to expand this into a CMS-ready template using something like Webflow or integrating it with a headless CMS like Sanity.
